ADEQUATE : Story: Task configurations may be listed

Created by Tomas Knap, last modified by Václav Brodec on Okt 05, 2016

Github Issues: https://github.com/odalic/odalic-ui/issues/105 , https://github.com/odalic/odalic-ui/issues/107

Req: 

User Story

As a user I would like to see list of configuration (in a table) from which I may select the one I would like to modify.

The order is from the most recently touched or updated to the least one. No filters are provided. The author column can be skipped for now.

The Task Name and Task Description cells are read only for now.



Preconditions

Detailed Description

  1. User may see list of task configurations. User may open detail for each, or run it or examine results.

Acceptance Criteria & Test Scenarios

  • The specified order is kept.
  • Controls work as intended (in other stories).
