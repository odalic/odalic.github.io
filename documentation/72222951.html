ADEQUATE : Story: Export resulting data as RDF data

Created by Josef Janoušek, last modified by Tomas Knap on Jul 13, 2016

Github Issue: 

User Story

As a user i want to export data as RDF, so that I can publish such data as Linked Data.

Preconditions

The user is logged on the system (see: Scenario 0a Login and home screen)

User have defined and run classification/disambiguation/relation discovery task (see: Scenario 1 Task creation). User is observing the results of the classification/disambiguation/relation discovery task. User reached the third step - relations specification. 


Detailed Description

  1. User clicks Export to RDF (see Scenario 4d) for the button's location. 

  2. As a result, system exports the given data (including the feedback provided) as RDF data (in Turtle representation). Together with the data, also CSV on the Web metadata is exported (JSON file). 

  3. User may download: 
    1. the RDF data file (turtle)

Acceptance Criteria & Test Scenarios

  1. Run Odalic Core task
  2. Ensure that there is subject column and at least one relation to one another column. 
  3. Click Export to RDF data
  4. Check that: 
    1. all files are included and properly formatted
