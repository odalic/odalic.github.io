ADEQUATE : Story: User can search KB for a concept to be used for classification/disambiguation/relation discovery

Created by Tomas Knap, last modified by Václav Brodec on Okt 05, 2016

Github Issue: https://github.com/odalic/sti/issues/87 / https://github.com/odalic/odalic-ui/issues/21 / https://github.com/odalic/sti/issues/189

User Story

As a user i want to be able to search KB for a resource to be used for classification/disambiguation. Also for relation discovery in a similar way!

There is autocomplete as well!

Preconditions

User can see results of classification/disambiguation (Story: User can see results of classification/disambiguation)

Detailed Description (Searching for alternative classification)

  1. User can click on "Customize" button next to the column header
    1. To see the location of that button, please see mockup in Scenario 4b Customizing classification class - selecting different non-suggested primary class for the column classification
    2. As a result, system shows the dialog
      1. The part of the dialog relevant for this story is depicted below
      2. The complete dialog is depicted in Scenario 4b Customizing classification class - selecting different non-suggested primary class for the column classification
      3. Below the suggested resources, system shows the search text field. (Mockup 1)
  2. User may start writing the name of the resources he is looking for (Mockup 2)
    1. As a result, system shows available resources (together with their URIs in the brackets)
  3. User may select one of the available resources (Mockup 2)
  4. User may click Add to add the shown resource to the list of (Mockup 3)
    1. As a result, dialog is closed and such resource is added to the list of classes. (Mockup 4)

 


Acceptance Criteria & Test Scenarios

  1. Run Odalic Core
  2. Customize classification
  3. Search for alternative class, try to add it. 
  4. Ensure that new class can be selected and after closing customize dialog also appear in the table
  5. Test also for disambiguations, relation  discovery

Variations

Searching for entity disambiguating the given cell value

The detailed description works similarly when searching for entity disambiguating the given cell value - the differences:  

  • Different title of the dialog - "Customize disambiguation"

 

Repeating the searching process

After Step 2/3, user may decide to tune the searched term by adjusting what is being searched
