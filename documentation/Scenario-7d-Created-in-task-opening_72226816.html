ADEQUATE : Scenario 7d Created in task opening

Created by Václav Brodec on Mai 21, 2016

Initial State

The user is logged to the application.

Description

  1. User enters the output file management screen.

  2. User clicks the open task button on the row with chosen file and the task creation dialog appears. (Mockup 1).

Alternatives

  • The task might not exist any more, so the "Created in" cell can be empty.

What can go wrong

  • The task might be removed before the page update.

System state on completion
