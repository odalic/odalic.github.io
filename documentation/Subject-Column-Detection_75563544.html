ADEQUATE : Subject Column Detection

Created by Tomas Knap, last modified on Sep 26, 2016

Currently, subject column detection is based on the characteristic of the table, so it should be the same for all KBs. When merging results of relation discovery from more KBs, subject column should be the same (If not, this would also not be a big issue - we can simply display all returned relations)

Subject column detection is relevant for relation discovery - it influences the column for which relations are searched.

 

Note: Based on the experience of Vasek, it happend to him that dbpedia and german dbpedia did not agree in certain cases on subject column!
