User Story

As a user I want to export the current task configuration to a file in order to allow me to make a backup of it or move the processing to different instance.

This should be available using a single button in the task configuration screen.

TODO: Decide on the file format.



Preconditions

  • All the parts of configuration are available and set.

Detailed Description

  1. User clicks the button for exporting the current task.
  2. A save file dialogue for the serialized task configuration appears.
  3. User chooses a location to save the file.
  4. File is downloaded from the server.

Acceptance Criteria & Test Scenarios

  1. Complete task configuration is exported in syntactically valid format.
  2. Application successfully handles export/import cycle.
