ADEQUATE : Story: Task configurations - last results may be explored

Created by Tomas Knap, last modified by Václav Brodec on Okt 05, 2016

Github Issues: https://github.com/odalic/odalic-ui/issues/105

Req: 

User Story

As a user I would like to see task result for the last execution of that task (Only last execution).




Preconditions

Detailed Description

  1. User may see list of task configurations. User may open detail for each, or run it or examine results.

Acceptance Criteria & Test Scenarios

  • Last status is periodically updated.
  • Actions reflect the current state of task and do not allow to run more than one instance of the task configuration at once.

Variations

  • When the task ends with an exception, it should be available to the user when he clicks on the status icon as pop-up.
