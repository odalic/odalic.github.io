ADEQUATE : Labels for titles of the external resources

Created by Tomas Knap, last modified on Sep 19, 2016

When resource are added via lodlive/lodview to Odalic UI (so that they can be used to annotate cells), the following predicates should be looked for (these predicates are recommended to contain title of a resource): 

  • rdfs:label
  • dcterms:title
  • skos:prefLabel
  • foaf:name
