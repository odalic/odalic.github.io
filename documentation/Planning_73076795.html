    • Note from project proposal:

      • System must save the configuration in machine readable format, e.g. JSON-LD, RDF data format serializations. Every such saved configuration has its own Linked Data identifier.

      • The configuration includes: • URL of the configuration (identifier), • description of the conversion, • configuration for Odalic Core algorithm (knowledge bases used, similarity metrics, feedback from the user), • inputs to which it was applied (if any), • settings of the outputted RDF data format (if available).

      • • The configuration must be complete, so that it is possible to execute the same conversion later in the future to the same/similar inputs.
