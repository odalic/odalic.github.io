ADEQUATE : UnifiedView DPU

Created by Tomas Knap, last modified on Feb 14, 2017

Inputs:

  • FileDataUnit 
    • input CSV file

Outputs:

  • 2xFilesDataUnit
    • Extended CSV file
    • RDF - Turtle

Idea: 

Use may past configuration exported from Odalic. User must also define URL of the Odalic server. 

As the DPU is executed, it automatically changes the configuration provided by user to generate unique fileID. Then it:

  • imports the task to the target Odalic server (defined in the conf) (TODO: what to do if there is already configuration with the same name?)
  • executes the task
  • tries to get result regularly - and show status
    • ContextUtils.sendInfo(), ContextUtils.sendError(), ...
  • produce output - 2 files
  • produce proper event messages - there should be one event message with the link to the "detail" where user can see the annotations provided by the server.
