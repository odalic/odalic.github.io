Odalic Core - Classification/Disambiguation/RelationDiscovery API

Description

This API is used by Odalic Server to run particular phases of the algorithm and provide information about the results to UI ( list of suggested winning concepts to be shown, etc. )

API

runClassificationDisambiguation(File f)

Runs the (re)classification and (re)disambiguation for the input file. Standard phase of Odalic Core.

Used by: Odalic Server


runRelationDiscovery(File f)

Runs the process of (re)discoverying relations for the given input file. Standard phase of Odalic Core.

Used by: Odalic Server


URL getWinningClass(Column c)

Returns the suggested winning concept classifying the given column.

Used by: Odalic Server


List<URLs, Score> getSuggestedClasses(Column c, int num)

Returns list of suggested concepts classifying the given column c, including also scores for such concepts. Optionally restricted by num, which specifies max number of such suggested concepts returned.


URL getWinningDisambigEntity(Cell c)

Returns the suggested entity disambiguating the given cell value.


List<URLs, Score> getSuggestedDisambEntities(Cell c, int num)

Returns list of resources classifying the given column c, including also scores for such resources. Optionally restricted by num


runClassificationDisambiguation(Column c)

Runs the classification for the given column c. This is called when re-classification is needed.


generateResultCsvData(File f)

Creates the resulting CSV file + metadata describing classification/disambiguation/relation discovery based on the original file and settings for classification, disambiguation, relation discovery (CSV on the Web standard)


generateResultingRdfData(File f)

Creates the resulting RDF file based  on the original file and settings for classification, disambiguation, relation discovery (CSV on the Web standard)


changeWinningConcept(Concept new)

Changes the winning suggested concept suggested by Odalic Core with a different suggested concept

Reqs: PP_07


selectNewWinningConcept(Concept new)

Changes the winning suggested concept suggested by Odalic Core with a new concept selected from the knowledge base

Reqs: PP_07
