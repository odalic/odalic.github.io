  • any23-sti
    • Anything To Triples (Any23) is a library, a web service and a command line tool that extracts structured data in RDF format from a variety
      of Web documents. This modules provides some extensions to that Any23 library (as Ziqi said, the original motivation for these extensions was to get the xpath of the extracted RDFa triples). These extensions are used to extract RDFa triples from some webpages, such as IMDB, MusicBrainz and used for example in: 
      • TableContextExtractorIMDB
