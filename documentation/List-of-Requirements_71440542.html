62 functionalUserUser can save the classified/disambiguation data also as CSV files, which are exported together with metadata

Metadata are prepared based on the standard CSV metadata on the Web (https://www.w3.org/2013/csvw/wiki/Main_Page)

Metadata: titles, descriptions taken from the concept classifying the column, links to other concepts classifying the column (from other data sources)

Motivation: As part of ADEQUATe, we work on QA metrics for CSV files. So Odalic should also produce CSV files.

Outputted CSV on the Web metadata should contain also delimiters.
